Crédits

Site réalisé avec des logiciels open-source.

  • Jekyll, le générateur de sites statiques et l'administration est réalisée grace à Netlify CMS
  • Twitter Bootstrap 3 pour le style
  • Tricoté avec les doigts avec Ubuntu comme système d'exploitation, The Gimp pour la retouche d'images, Inkscape pour le dessin vectoriel, Sublime Text comme éditeur de code et bien d'autres encore.
